# project-one
Group Project One

https://hazelreyes.github.io/project-one/

(assets/images/project-one.jpg)

`project-one` is a JavaScript & jQuery web application that uses the Yelp and YouTube API's, Local Storage, and a new library Materialize. The user inputs a destination city with the results being top five restaurants in that city and a YouTube video of a restaurant in that city. The user can then add a restaurant to the itinerary by clicking on the restaurant of choice. The user will also be able to delete restaurants on the itinerary.

## Purpose

To use the JavaScript programming language and jQuery library to create a web application that uses at least two API's, a method of storing information, and a new programming library.

## Goals

- Create a web application using Javascript and jQuery library
- Use Local Storage
- Use Yelp API
- Use YouTube API
- Use jQuery methods
- Use functions
- Use if/else statements
- Use Materialize for CSS stylings
- Use modals
- Incorporate input validation
- Practice Agile software development
- Creating an MVP
- Practice Git branching

## Instructions

1. Install Visual Studio Code
1. Install Git Bash
1. Create repository `project-one` in GitHub
1. Use Git Bash to `git clone` the repo `project-one`
1. Create `index.html` file and `assets` folder
1. Inside `assets` folder, create folders `css` and `javascript`
1. Inside the `css` folder, create `reset.css` file
1. Inside the `javascript` folder, create `game.js` file
1. `git add index.html`
1. `git commit -m` with description inside quotation marks
1. `git push origin master`
1. `git add assets/*`
1. `git commit -m` with description inside quotation marks
1. `git push origin master`
1. Commit changes to repository
1. Deploy repository to hosting service
